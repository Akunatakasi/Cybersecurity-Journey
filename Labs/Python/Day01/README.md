## VS Code: Select Interpreter
1. Open the Command Palette with Ctrl+Shift+P.
2. Run Python: Select Interpreter.
3. Choose the installed Python version, such as Python 3.x.
4. Confirm the selected interpreter appears in the bottom-right status bar.

If no interpreter is listed, install Python and reload VS Code.
