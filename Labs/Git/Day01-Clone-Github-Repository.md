# Day 01 - Cloned My GitHub Repository

## Goal
Set up my Cybersecurity-Journey repository on my laptop.

## Process

1. Opened VS Code.
2. Installed Git and confirmed it was working.
3. Signed in to my GitHub account from VS Code.
4. Opened the Command Palette (`Ctrl + Shift + P`).
5. Selected **Git: Clone** (or **Clone Repository**).
6. Pasted my GitHub repository URL:
   ```bash
   https://github.com/Akunatakasi/Cybersecurity-Journey.git
   ```
7. Selected a location on my **D:** drive to save the project.
8. Opened the cloned repository in VS Code.
9. Verified everything was downloaded successfully.

## First Git Commands

```bash
git status
git add .
git commit -m "First commit"
git push origin main
```

## Lesson
- `git clone` downloads a repository from GitHub to a local computer.
- VS Code is where I write and edit code locally.
- Git tracks changes locally.
- GitHub stores and backs up my work online.
