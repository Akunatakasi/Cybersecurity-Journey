# Day 01 - Connected GitHub to VS Code

## Goal
Set up my Cybersecurity-Journey repository in VS Code and connect it to GitHub.

## What I Did
- Signed into GitHub from VS Code.
- Opened my local `Cybersecurity-Journey` folder.
- Initialized Git (if needed).
- Connected the folder to my GitHub repository.
- Confirmed the repository was linked.
- Learned to use:
  ```bash
  git status
  git add .
  git commit -m "message"
  git push
  ```
- Successfully pushed my first updates to GitHub.

## Lesson
- VS Code is my local workspace.
- Git tracks changes locally.
- GitHub stores my project online.
- My workflow is:
  **Edit → git status → git add . → git commit → git push**
